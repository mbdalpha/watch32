Pin Summary Table
-----------------

  ESP32 GPIO    Display Pin    Function
  ----------    -----------    --------
  GPIO 13       D0             Data bit 0
  GPIO 12       D1             Data bit 1
  GPIO 14       D2             Data bit 2
  GPIO 27       D3             Data bit 3
  GPIO 26       D4             Data bit 4
  GPIO 25       D5             Data bit 5
  GPIO 33       D6             Data bit 6
  GND           D7             MUST be tied to GND (floating causes noise)
  GPIO 15       Ad0            Address bit 0 (digit select)
  GPIO  2       Ad1            Address bit 1 (digit select)
  GPIO  4       Ad2            Address bit 2 (digit select)
  GPIO 18       CE             Chip Enable (active LOW)
  GPIO 19       RST            Reset (active LOW)
  3V3           VCC            Power supply
  GND           GND            Ground


Signal Timing
-------------

    Address  --| VALID |------------------
                    | setup time (~1us)
    Data     ------| VALID |--------------
                        | setup time (~1us)
    CE       ------------\     /----------
                          \---/
                          | pulse (~1us)


Notes
-----

  - D7 must be tied to GND. Leaving it floating causes display noise.
    Setting D7 HIGH enables custom character mode.

  - 3.3V is fine. The display works at 3.3V from the ESP32, no level
    shifter needed.

  - No resistors needed. The smart display has built-in current limiting.

  - 12 GPIO pins total. For a custom PCB you could reduce this to 3-4
    pins by adding a shift register (e.g. 74HC595) to handle the
    data/address bus, saving board space.

  - Avoid ESP32 GPIO 6-11 (connected to flash) and GPIO 34-39 (input only).
